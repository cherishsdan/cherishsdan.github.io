About AngularJS

  • angular

昨天面试，面试官问了很多angularJS的问题，比较深入，对于我说的不清楚的，也给了解答，总的来说，一次很愉快的面试体验，收货很大

  • angularJS的路由原理
    ui-router vs ngRouter
    1、多视图（页面可以显示多个动态变化的不同区块）
    在ngRoute中：
视图没有名字进行唯一标志，所以它们被同等的处理。
路由配置只有一个模板，无法配置多个。
在ui.router中：
可以给视图命名，如：ui-view="status"。
可以在路由配置中根据视图名字(如：status)，配置不同的模板（其实还有controller等）。
    2、嵌套视图(页面某个动态变化区块中，嵌套着另一个可以动态变化的区块)
    ngRouter:没有明确的父子层级关系
ui.router:通过parent与parent.child来确定路由的父子关系，从而解决无限递归问题。
另外子路由的模板最终也将被插入到父路由模板的div[ui-view]中去，从而达到视图嵌套的效果。
    ui-router的工作原理：一个查找匹配的过程
    more
    由于我在项目中使用的是ui-router,所以主要讲的ui-router
    先引入：
    angular.module("myApp", ["ui.router"]);
    这样做的目的是，在程序启动(bootstrap)的时候，加载依赖模块(如：ui.router)，将所有挂载在该模块的服务(provider)，指令(directive)，过滤器(filter)等都进行注册，那么在后面的程序中便可以调用了。
