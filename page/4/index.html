CSS

  • 盒子模型
    盒模型指定元素如何显示，页面上的每个元素都被看成一个矩形框，这个框由元素的内容、内边距、边框和外边距组成。

前端性能优化——载入

发表于 2016-07-05   |   分类于 learning~   |     |   阅读次数

性能优化系列笔记，提升用户体验

从Yahoo军规到现在各种各样的优化技术，不能一概而论，还是要根据用户需求，
制定适合自己的优化方案

载入优化，简单来说，就是重要的同步加载，不重要的异步加载。
以FaceBook的BigPipe方案为例，它大概有3步：

  1. 首屏先同步加载，返回一些占位符
  2. 根据需求，请求占位符的资源（ID）
  3. 填充占位符

不过这种方案也有缺点:

  1. SEO 引擎较难抓取
  2. 异步请求较多
  3. 模块之间相互通信引用
  4. 模板引擎渲染
